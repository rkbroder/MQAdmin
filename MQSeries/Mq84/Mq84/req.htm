How to check if G2 is installed

Netscape users
  • Select Help from the top bar menu
  • Select 'About Plug-ins'
  • Scroll down trying to find the words RealPlayer(tm) G2 LiveConnect-Enabled Plug-In (32-bit)

What to do if it has not been installed
  • Close this window
  • Download RealAudio G2 player
  • Make note of the folder in which G2 player file is saved
  • Close Browser
  • From Windows Explorer, double click the downloaded file
